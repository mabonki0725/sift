##H2 SIFT(Scale Invariant Feature Transform)特徴量の計算

_処理_ 
1. P3 or P2 PPMファイルの読込み  
2. Gussian Filterの異なるσ幅での画像の平滑化  
3. 隣合うσ幅の平滑化画像の差を特徴点とする  
4. 特徴点の曲率が閾値以下なら削除（点列と判断）
5. 特徴点の極大値が閾値以上なら採用  
6. 特徴点間の強度と方向を計算、強度が閾値以上なら採用
